Estrutura de repetição com teste lógico no início

        var c = 1
        while (c <= 10) {
            console.log(`Passo ${c}`)
            c++ // var c recebe + 1
        }

Estrutura de repetição com teste lógico no final

        var c = 1
        do {
            console.log(`Passo ${c}`)
            c++ // var c recebe + 1
        } while (c <= 20)

Estrutura de repetição com variável de controle

            for (inicio ; teste; incremento) {

            }

            ou seja

            for (c = 1 ; c <= 10 ; c++) {
                console.log(`Passo ${c}`)
            }
