Operadores Relacionais

5 > 2 == true

7 < 4 == false

8 >= 8 == true

9 <= 7 == false

5 == 5 == true

4 != 4 == false

5 == '5' == true | o JS não considera o tipo

Identidade

IGUALDADE RESTRITA

5 === '5' == false | para considerar o tipo de dado em uma relação, utilize '==='

5 === 5 == true

5 !== 2 == false | para considerar o tipo numa relação de diferenla usar !==

Operadores Lógicos

! == negação (not)

&& == conjunção (and)

|| == disjunção (or)

Ordem de execução

1º !(negação)

2º &&(conjunção)

3º ||(disjunção)

Precedência geral

1º Operadores aritiméticos ((), **, /...)

2º Operadores relacionais (>, <, >=...)

3º Operadores lógicos (!, &&, ||)

Operador ternário (três operandos)

teste lógico ? true : false

média >= 7.0 ? "Aprovado" : "Reprovado"

O teste lógico é o IF do Python.
