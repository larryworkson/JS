Operadores aritméticos

5 + 2 = 7

5 - 2 = 3

5 * 2 = 10

5 / 2 = 2.5 (divisão real)

5 % 2 = 1 (resto da divisão)

5 ** 2 = 25 (potência)

Ordem de Precedência

() > ** > *, / ou % > - +

Sempre da esquerda para direita

Simplificando atribuições

var n = 3

n = n + 4 == n += 4

n = n - 5 == n -= 5

n = n * 4 == n *= 4

n = n / 2 == n /= 2

n = n ** 2 == n **= 2

n = n % 5 == n %= 5

Operadores de incremento

n += 1 == ++n | a variável recebe + 1

n -= 1 == --n | a variável recebe - 1
